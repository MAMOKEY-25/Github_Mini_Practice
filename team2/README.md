## 💻 Git 컨벤션

`[ 타입 ] `  : `[ 간결한 설명 ]` + `[ #Issue 번호 ]`

- `feat`     : `회원가입 기능 구현`
- `fix`      : `이메일 인증 오류 수정`
- `docs`     : `README 수정`
- `style`    : `들여쓰기 수정 , 세미콜론 등 추가`
- `refactor` : `코드 구조 개선 ( 기능 변경 없음 )`
- `test`     : `로그인 기능 테스트 코드 추가`
- `hotfix`   : `긴급 수정`
